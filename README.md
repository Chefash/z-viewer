## What It Does

**Z-Viewer instantly shows how private your Zcash wallet is** — with a **Privacy Score (0–100%)**, **live chart**, and **transaction breakdown**.

- **100% shielded demo** loads automatically  
- **No API failures** — works offline  
- **One-click "Share on X"** to go viral  
- **Zero server storage** — all in your browser

## How to Use

1. Open [zviewer.netlify.app](https://zviewer.netlify.app)
2. **Instant demo loads** (100% shielded)
3. Paste any **Zcash address** (t1... or zs1...)
4. Click **"View Privacy"**
5. See your **Privacy Score** + chart
6. Click **"Share on X"** to flex your privacy
